CNC Conversion Project

The CNC Mill when I receieved it had a traditional control system, using two fading 8 inch CRT monitors, making it almost impossible to use in any reasonable (safe) lighting conditions. In addition, only drip-fed GCode, or manually remade code on the internal computer was possible.

I decided to kill two birds with one stone and upgrade the computer to a dedicated linux box, which is connected by wifi to Dropbox, a easy storage solution for getting GCode to the computer. It currently uses a high resolution 20 inch screen, and is fully self contained.
